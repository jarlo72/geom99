GEOM99 Group Project

Topic Selection

The topic that was agreed upon is creating a Storymap and website based on the Vancouver Island Trail System and information on Vancouver Island in British Columbia, built and monitored by the Vancouver Island Trail Association (aka VITA for short). The reason we've chose this topic was some of us in the group had a similar direction in developing a web map from the provided data constructed via ArcGIS Pro using doing analysis on them; same goes for the rest of the group as they are doing something similar with ArcGIS Pro for their collaboration project. Therefore, the topic was a good match to move forward.

Gathering and Processing Data

Firstly, the data was extracted from the VITA's website as they had the GPX files in store for use.
